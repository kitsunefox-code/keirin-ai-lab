# 競輪の基礎知識ノート

出典: KEIRIN.JP 競輪ガイド(https://keirin.jp/pc/static/beginner/)。
アプリの展開予想・モーション再生・シナリオ文はこのノートの定義に従う。

## バンク(競走路)の名称

- **ホーム・ストレッチ・ライン**: ホームストレート上のライン。ゴール線でもある。残り1周のホーム線を先頭通過した選手にホーム数(H)が加算される。
- **バック・ストレッチ・ライン**: 決勝線からちょうど半周の位置。残り半周時点で先頭通過した選手にバック数(B)が加算される。
- **イエロー・ライン**: バンク中央やや下方の3cm幅の黄色い線。先頭選手がこの外側を一定時間走ると失格の場合がある。
- **外帯線**: 内側2本の線のうち外側。外帯線の内側を前走する選手を内側から追い抜くと「内側追い抜き」で違反。
- **内圏線**: 内側2本の線のうち内側。
- **退避路**: 内圏線の内側の平坦な茶色い部分。退避時に走行する。
- **30m線**: ゴール手前30m。最終周回でここを越えて落車した場合、自転車を担いでゴールしても完走扱い。
- **25m線**: 発走線から25m。ここまでの競走が不適当なら再発走。
- コーナーは1〜4コーナー、その間に1センター(1-2間)・2センター(3-4間)。

## レースの流れ

1. **選手紹介(脚見せ/顔見せ/地乗り)**: どのラインを組むかをアピールする。並び確認の重要情報。
2. **スタート**: 発走機に後輪固定、号砲で一斉スタート。まず先頭誘導員(先頭員)の後ろを走る。S(スタート回数)=号砲後すみやかに先頭員の後方についた回数。
3. **周回**: 先頭員が風除け。残り3周=青板、残り2周=赤板で表示。
4. **打鐘(ジャン)**: ゴールまで**残り1周半**のバック・ストレッチ・ラインを先頭選手が通過すると鳴る。ここからピッチが上がる。
5. **ゴール**: 最後の直線は時速70km近い。

## ライン

- ラインは脚質を活かして組むチーム。基本形は「先行/捲り(自力型)」の後ろに「追込み(他力型)」。
- 登録地(地区)の近い選手と組むのが基本。
- **戦型**: ライン数で呼ぶ。
  - 2分戦: 2ライン(5人・4人など、ラインが長い)
  - 3分戦: 3ライン(3・3・3、4・3・2など)
  - 4分戦: 4ライン(3・2・2・2など)。短いラインが多く「細切れ戦」とも呼ぶ
  - 先行1車: 自力型が1人だけの展開
- **番手(ラインの2番手)の役割**: 主導権争いのサポート。
  - ブロック: 後方から捲ってくる選手を止める
  - 車間をあける: 追い上げ距離を伸ばす
  - 切り替え: 先行・捲られた際に他ラインへ追走を替える
  - 競り: 強い先行選手の番手を取り合うこと

## 脚質と戦法

- 地脚タイプ: 持久力型。スプリンタータイプ: ダッシュ型。
- **先行**: 先頭を走る戦法。
  - 抑え先行: 前の先行選手まで上がって抑え、逃げ切れるタイミングで駆ける
  - つっぱり先行: 抑えに来られても内から強引に先行を譲らない
  - かまし先行: 前がスピードを上げないうちに一気に踏み上げて先行
- **追込み**: 先行選手の後方につけ、最終直線で差す。
- **捲り**: 終盤に別ラインの先行選手を後方から一気に追い抜く。トップスピードとダッシュ力が必要。
- けん制/ブロック: 後方から仕掛ける選手を止める動き。

## 出走表の読み方(WINTICKET/KEIRIN.JP共通の概念)

- 決まり手: 逃(逃げて1-2着)・捲(捲って1-2着)・差(ゴール前追込みで1-2着)・マ(マークで2着)
- B=バック回数(残り半周先頭通過)。終盤の展開予想に最重要。
- H=ホーム回数(残り1周先頭通過)。勝負どころで前にいる傾向。
- S=スタート回数(先頭員の直後を取った回数)。位置取りの積極性。
- 勝率/2連対率/3連対率: 直近4ヶ月。
- 直近成績の記号: 数字=負け戦系、丸数字=勝ち上がり、黒丸数字=特選/決勝系、欠=欠場、落=落車、棄=棄権、失=失格。

## 他のAI予想家から学んだ手法(2026-07 リサーチ)

出典: Google検索 + note「パ部競輪」(過去6年半・15万レースを機械学習した個人AI開発者)、netkeirin「ウマい車券 AIライン」、WINTICKETコラム。

- **番手の優位性**: 番手選手は風の抵抗を受けずスタミナ温存できるため、AIでも最も1着になりやすいポジションとして評価するのが定石。
- **2軸ライン強度**(パ部競輪): ①ライン先頭選手の競走得点でラインをランク付け(A1=最強ライン先頭, A2=その番手...) ②先頭選手のバック回数で第2のランク付け(a/b/c/d)。得点だけだと本命寄りになり6・8番車の逃げ/捲りを取りこぼすため、バック回数軸で穴目を残す。「的中率を上げるにはライン強度をうまく特徴量にするのが近道」。
- **勝利確率ベース**(netkeirin AIライン): 3連対予測率ではなく1着の「予測勝利確率」を本軸にし、小点数構成へ変更したことで回収率が10%以上向上。
- **オッズ・人気は特徴量に入れない**(パ部競輪): 市場の後追いになるため予測には使わない(期待値計算にのみ使う)。
- **バンク特性**: 周長(333/400/500)・カント・みなし直線距離で有利な脚質が変わる。選手ごとの周長別成績・当該バンク成績を見る。

### アプリへの実装対応

- WINTICKET recordsの `linePositionFirst/Second/Third/lineSingleHorseman` = 先頭勝率・番手勝率・3番手勝率・単騎成績 → 今回のライン位置に対応する成績を特徴量+加点に使用。
- `latestVenueResults` = 当該バンクでの過去成績(当所成績)、`trackDistance333/400/500` = 周長別成績、`hourType*` = 時間帯別成績 → バンク適性として使用。開催バンクの周長・直線・カントは venues データから取得。
- 2軸ライン強度を `_attach_line_ranks`(predictor.py)で実装。バック数最強ラインの先頭・番手は「穴目」として根拠に表示。

## 競輪場・バンクのクセ(2026-07 WINTICKETバンク解説より)

各競輪場のバンク特徴はWINTICKETの出走表に付随するバンク解説(bankFeature)＋決まり手分布(factors)から
venuesテーブルへ記憶する(場ごとに1度、出走表1本で全場ぶん取得できる)。

### 周長による傾向(実データで確認)
- **333m(短走路)**: 逃げ・先行が有利。コーナーが多く直線が短いため先行選手が粘りやすい。
  例: 奈良 bias+0.75 / 松戸 +0.69 / 小田原 +0.61 / 前橋 +0.50。
- **400m(標準)**: 脚質の有利不利は小さめ。やや差し寄りの場が多い。
  例: 久留米 bias-0.21(逃26/捲28/差46) / 函館 -0.24。
- **500m(長走路)**: 差し・追込が有利。直線が長く先行では最後まで持たない。
  例: 大宮 bias-1.0(逃20/捲25/差55) / 宇都宮 -1.0 / 高知 -0.9。

### 数値化(bank_bias)
決まり手分布(逃げ%-差し%)を主軸に、周長が短い/直線が短いほど先行有利へ補正した値を
-1(差し・追込有利)〜+1(逃げ・先行有利)で持つ。脚質(逃=+1/両=+0.3/追=-1)との積が
正なら適合として加点、負なら減点する。

## ナイター・ミッドナイト・天気

- **ナイター**: 夕方〜夜開催。選手の時間帯別成績(hourType)を個人適性として見る。
- **ミッドナイト競輪**: 深夜開催で基本**無観客**。屋内(isIndoor)で行うことも多く、
  出走が7車立てでなくA級中心・特殊な展開になりやすい。先行有利の傾向を軽く加味する。
- **天気(雨)**: 路面が滑り差しづらくなるため、逃げ・先行が残りやすい。捲りは決まりにくい。
  WINTICKETの天気コード(100晴/200曇/300雨/400雪)と降水量で判定する。

## レース級班

- **S級**: 上位クラス。GI/GII等では実力・近況の比重が高い。
- **A級**: 中間クラス。
- **ガールズ(L級)**: 女子選手。**単騎主体でライン概念が薄い**ため、ライン強度の重みを下げ、
  個人の競走得点・勝率・出走選手同士の対戦成績を重視する。

## 違反(失格になり得る代表例)

- 斜行(進路妨害)、押圧(内へ押し込む)、内側追い抜き、押し上げ(外へ押し上げる)。
- 失格は着位剥奪。車券にも直結する。
